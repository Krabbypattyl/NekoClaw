# NekoClaw

> bese on CoPaw

NekoClaw 是一个基于 CoPaw 定制的个人 AI 助手项目。

## 快速开始

完成依赖安装和初始化后，运行：

```bash
nekoclaw init --defaults
nekoclaw app
```

如果想使用交互式初始化：

```bash
nekoclaw init
nekoclaw app
```

## 常用命令

```bash
nekoclaw app
nekoclaw init
nekoclaw uninstall
nekoclaw --version
```

## 说明

- 对外显示名：`NekoClaw`
- 命令行入口：`nekoclaw`
- 当前项目为基于 CoPaw 的定制版本
