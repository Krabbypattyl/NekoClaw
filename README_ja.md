# NekoClaw

> bese on CoPaw

NekoClaw は CoPaw をベースにカスタマイズした個人向け AI アシスタントです。

## クイックスタート

依存関係のセットアップ後、以下を実行します。

```bash
nekoclaw init --defaults
nekoclaw app
```

対話形式で初期設定したい場合：

```bash
nekoclaw init
nekoclaw app
```

## よく使うコマンド

```bash
nekoclaw app
nekoclaw init
nekoclaw uninstall
nekoclaw --version
```

## メモ

- 表示名: `NekoClaw`
- CLI コマンド: `nekoclaw`
- このプロジェクトは CoPaw ベースのカスタム版です
