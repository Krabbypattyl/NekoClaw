# NekoClaw

> bese on CoPaw

NekoClaw is a personal AI assistant project customized from CoPaw.

## Quick Start

Install dependencies and set up the project first, then run:

```bash
nekoclaw init --defaults
nekoclaw app
```

If you want to use the interactive setup flow:

```bash
nekoclaw init
nekoclaw app
```

## Common Commands

```bash
nekoclaw app
nekoclaw init
nekoclaw uninstall
nekoclaw --version
```

## Notes

- Display name: `NekoClaw`
- CLI command: `nekoclaw`
- This project is currently maintained as a customized version based on CoPaw
